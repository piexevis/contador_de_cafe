# contador de café
Aplicación de escritorio simple hecha en Java con Swing.
Cuenta cuántos cafés tomaste y te da mensajes según la cantidad.

## Tecnologías

- Java y su libreria Swing

## Qué aplica

- Herencia (JFrame)
- Encapsulamiento (simple pero sigue siendo su encapsulamiento)
- Lambda (la webada esta "->")
- Switch (con la webada esta "->")
